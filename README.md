### Installation

```
composer create-project labile/astaroth-framework bot
```

### Requirement

> Платформа - `Linux`\
> Версия `PHP` - `>=8`\
> Расширения: `ext-curl`, `ext-pcntl`, `ext-posix`, `mbstring`\
> Веб сервер для callback: `nginx`, `apache`
